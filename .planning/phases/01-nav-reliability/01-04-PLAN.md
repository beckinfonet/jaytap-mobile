Hard go/no-go gate. The human re-runs the full reproduction matrix on the Wave-1 build (Plan 03 commits landed), collects [NAV] diagnostic logs during the canonical trap sequence, and records results in the matrix's Post-Wave-1 section. The decision at the end of this plan is binary:

- PROCEED-TO-WAVE-4: if every cell is PASS (0% repro, zero FAIL, zero FLAKY). Plan 05 escalation skipped; Plan 06 strip + finalize runs next.
- ESCALATE-TO-WAVE-3: if any cell remains FAIL. Per D-04, if the baseline was 100% FAIL on a cell and Post-Wave-1 is now PASS on the same cell while other cells remain FAIL, that pattern is evidence the root-cause model is wrong — human and Claude pause, reassess, re-discuss BEFORE launching Plan 05. If the D-04 condition does not trigger, Plan 05 launches the C2 → C3 → C4 escalation sequence.

Purpose: NAV-01 + NAV-02. This is the decider between "ship clean on Plan 06" and "escalate diagnostics." The gate is encoded per D-04: partial success is treated as evidence the model is wrong, not a win.
